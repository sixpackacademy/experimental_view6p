Introdução
O nosso projeto final da disciplina PWC (Programação Web-Cliente) baseia-se na criação do website do ginásio local chamado sixpackacademy. O ginásio não tem website, portanto decidimos realizar um website em que os utilizadores possam marcar o seguinte serviço:
    * Fisioterapia desportiva
    * Fisioterapia
    * Reabilitação desportiva
    * Avaliação postural
    * Massagem de relaxamento
    * Massagem descontrotaturante
    * Drenagem linfática
    * Knesio taping

O administrador pode aceitar ou recusar as marcações, quando aceitar a marcação, o cliente irá receber uma notificação a dizer que foi marcado.

Para trabalharmos em grupo, usamos um sistema de versão de controlo git. Com a ajuda do github desktop, conseguimos colaborar juntos e fazer o projeto em simultâneo.
O nosso website vai ser feito com HTML, CSS e JavaScript. Também usamos a framework Bootstrap
